# Tema Example

Example application demonstrating the usage of the **tema** package.

## Running the Example

```bash
cd packages_dashboard/tema/example
flutter pub get
flutter run -d chrome
```

## Features Demonstrated

- Theme mode selection (Light, Dark, System)
- Theme toggling
- Theme persistence across app restarts
- System theme detection
- Component styling preview
- Color palette display

## Screenshots

The example app showcases:
- Current theme status display
- Radio buttons for theme selection
- Quick toggle button in AppBar and FAB
- Material Design 3 component examples
- Color palette visualization

## Usage

The example demonstrates all the features of the tema package:

1. **Theme Selection**: Use radio buttons to select light, dark, or system theme
2. **Quick Toggle**: Use the FAB or AppBar button to quickly switch themes
3. **Component Preview**: See how different Material components look in each theme
4. **Color Palette**: View the color scheme being used

## Code Structure

- `main.dart`: Complete example application with all features
